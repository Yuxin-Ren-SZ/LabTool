LabTools

Interactive utilities for the cell biology bench. Open source, no install required.

Made by Yuxin with Claude and GitHub Copilot

Available Tools
🧮
Cell Count Calculator
Hemocytometer counting with Trypan Blue viability analysis. Supports single or duplicate chamber counts across 4 counting modes.
Live Open →
⚗️
Cell Seeding Calculator
Count your cells, then calculate the dilution needed to reach a target seeding density. Guided two-step workflow with C₁V₁ = C₂V₂ solver.
Live Open →
🧫
Well Plate Map Generator
Assign treatment groups to 6-, 12-, 24-, 48-, 96-, 384-, or 1536-well plates, copy patterns across wells, and export CSV or print-ready layouts.
Live Open →
⏱
Stain Timer
Configurable step-by-step staining protocol timer with alarm. Design your protocol, then run it with countdown and audio cues.
Live Open →
🏷️
Thermal To Laser Label Converter
Convert a one-label-per-page thermal printer PDF into a mailing-label-sheet PDF for laser printing, with preset detection, sheet preview, and cell-by-cell placement control.
Live Open →
