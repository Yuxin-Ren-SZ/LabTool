1. Source PDF
Select one or more PDFs. Supporting browsers in secure contexts will start in Downloads; other browsers use the normal file picker. Files are combined in the order you choose them.
Upload one or more thermal-printer PDFs to begin.
2. Preset And Sheet Geometry
Built-in presets ship from tools/thermal-to-laser/preset-config.js. Browser-saved presets stay local until you copy or export them back into that file.
Saved presets stay only in this browser’s local storage until you clear them. They do not sync to other browsers or machines, and they do not rewrite tools/thermal-to-laser/preset-config.js. Use “Copy Current Settings” or “Export All Presets” to merge changes back into the repo file manually.
3. Sheet Layout
“Start Here” resets the full plan and fills cells sequentially from the clicked position. “Use Cell” moves a label into that cell. “Skip Cell” marks a cell unavailable and advances the next pending label to the next open slot.
The preview will auto-fill from the first cell when the preset becomes valid.
4. Export
Generate the PDF after reviewing the sheet preview. Supporting browsers will prefer Downloads when saving; other browsers use their default download flow.
